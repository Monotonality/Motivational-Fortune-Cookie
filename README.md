# Fortune Cookie Web App

A simple and elegant web application featuring a fortune cookie that reveals motivational fortunes when clicked. Built with a beautiful Chinese New Year color palette.

## Features

- **Interactive Fortune Cookie**: Click the cookie image to reveal your fortune
- **Beautiful Design**: Chinese New Year inspired color scheme with reds, golds, and dark accents
- **Responsive Layout**: Works perfectly on both desktop and mobile devices
- **Smooth Animations**: Elegant transitions and hover effects
- **100+ Fortunes**: Extensive collection of motivational and inspiring fortune messages
- **Reset Functionality**: Get a new fortune anytime with the reset button

## Color Palette

The app uses a traditional Chinese New Year color scheme:
- **Primary Red**: #8B0000 (Dark Red)
- **Accent Red**: #DC143C (Crimson)
- **Gold**: #FFD700 (Golden Yellow)
- **Dark Brown**: #2F1B14 (Deep Brown)
- **White**: #FFFFFF (Pure White)

## How to Use

1. **Open the App**: Simply open `index.html` in your web browser
2. **Click the Cookie**: Click on the fortune cookie image to reveal your fortune
3. **Read Your Fortune**: A beautiful golden fortune message will appear
4. **Get Another Fortune**: Click the "Get Another Fortune" button to reveal a new fortune

## Technical Details

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **Vanilla JavaScript**: No external dependencies, pure JavaScript functionality
- **Responsive Design**: Mobile-first approach with media queries
- **CSS Animations**: Smooth transitions and keyframe animations

## Browser Compatibility

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Getting Started

1. Clone or download this repository
2. Ensure all files are in the same directory
3. Open `index.html` in your web browser
4. Enjoy your fortune cookie experience!

## Customization

You can easily customize the app by:
- Adding more fortunes to the `fortunes` array in `script.js`
- Changing colors in `styles.css` to match your preferred theme
- Modifying the cookie image by replacing `CookieOfTruth.jpg`
- Adjusting animations and transitions in the CSS
